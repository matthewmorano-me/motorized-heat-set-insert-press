# ME106 Spring 2026 Group Project: Motorized Heat Set Insert Press
Team Members: Hector Barahona, Amin Elizondo, Dan Le, Matthew Morano
---
**Background**

Heat-set inserts are a type of threaded fastener that is permanently installed into thermoplastic materials, such as 3D-printed parts. They provide a strong threaded interface for installing screws to secure plastic parts. To install heat-set inserts, a soldering iron is typically used to heat up the insert and press it into the plastic part. However, manual installation with a soldering iron can result in inconsistent insertion depth, alignment, and variability in the quality of installation depending on the operator.

This project aims to design and develop a motorized heat-set insert press to improve the repeatability and consistency in installing heat-set inserts. By using a force and displacement sensors with a motorized vertical actuator, the system improves the installation process for heat-set inserts by reducing human variability and making the process more repeatable at scale.
